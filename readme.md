# Auto.js DevTools

## 环境配置

本工具依赖于[Node.js](https://nodejs.org)，并且只有通过Node.js内置的npm指令才可安装。

Node.js安装完成后，运行：

```bash
npm i -g autojs-dev
```

支持的指令：
```
autojs create <filename> # 在当前目录创建filename.auto.ts，使用'-j, --javascript'指令可创建filename.auto.js

autojs cap [filename] # 在当前目录创建filename.png，可通过'-d, --dir'指令设置保存目录
```
